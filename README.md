# homebrew-tap

Homebrew formulae for [virgoC0der](https://github.com/virgoC0der)'s tools.

## Usage

```bash
brew install virgoC0der/tap/zed-claude-bridge
brew services start zed-claude-bridge
```

## Formulae

| Formula | Description |
|---------|-------------|
| [`zed-claude-bridge`](Formula/zed-claude-bridge.rb) | Claude Code `/ide` integration for [Zed](https://zed.dev): at-mentions, automatic active-file & selection awareness, `openFile` jumps. See [claude-code-zed](https://github.com/virgoC0der/claude-code-zed). |
